## Plone développeur
L'objectif de cette formation est de vous apporter les compétences pour maîtriser le développement sur Plone.

### Objectifs
   * Maîtriser le développement sur Plone
   * Développer des produits Plone sur mesure

### Pré-requis
   * Connaître le langage Python
   * Connaître Plone

### Public
  * Développeurs

### Durée
* 4 jours

### Prix
* 2 000 €

### Programme
1. Zope Component Architecture
    * Les Interfaces
    * ZCML
    * zope.schema
    * Les Adapters
    * Les Adapters nommés
    * Les multi-Adapters
    * Les Factories
    * Les Utilities
    * Les Events
2. Les skins
    * Les browser_views
    * Les viewlets
    * Les portlets
3. Plone et les tests
    * Unittests / doctests
    * Tests d’intégration / fonctionnels
    * Selenium / Robotframework
4. Création d'un produit
    * plone.app (portal)
    * Les Tools
    * Archetypes / Dexterity
    * Les formulaires z3cform
    * Les widgets z3cform
    * Les Vocabularies
    * Les Permissions
    * Les ressources (JS/CSS)
5. Generic_setup
    * Import/export steps
    * Content_Types
    * Workflows
    * Skins
6. Optimisationge
